# Mito Peptides Humanin MOTS c

Mitochondria-encoded peptides. Humanin (16S rRNA): anti-apoptotic, neuroprotective, anti-inflammatory. MOTS-c (12S rRNA): metabolic regulation, exercise-mimetic. Emerging research; minimal autism work yet.
