# Patient profiles

This directory is for atlas-derived patient profile markdown files
generated by the upload + report flow on
[causes-atlas-autism.vercel.app](https://causes-atlas-autism.vercel.app/living_graph.html).

When a user clicks "Download Obsidian profile" in the report, the
resulting `atlas-profile-DATE.md` file goes here. Each one contains
wikilinks back into the atlas vault — open in Obsidian and the
graph view renders the patient's matched sub-network.

Patient data lives client-side (browser localStorage) by default.
Files dropped here are private to this vault.
