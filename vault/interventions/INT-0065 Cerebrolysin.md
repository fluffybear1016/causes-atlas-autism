# INT-0065 Cerebrolysin


Porcine brain-derived peptide mixture; trophic effects via neurotrophins. Used off-label in some pediatric autism protocols (Eastern Europe).


**Dose range:** 1-5 mL IM, courses of 10-20 doses


**Cost per month (USD):** 300


**Notes / warnings:** Pediatric autism trials small; off-label use widespread outside US.
