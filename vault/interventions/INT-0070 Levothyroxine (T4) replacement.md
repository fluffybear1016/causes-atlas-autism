# INT-0070 Levothyroxine (T4) replacement


Standard thyroid hormone replacement; restores fetal/childhood thyroid hormone in confirmed hypothyroidism.


**Dose range:** 1.6 mcg/kg/day adult; pediatric weight-based


**Cost per month (USD):** 15


**Notes / warnings:** Only when hypothyroidism documented.
