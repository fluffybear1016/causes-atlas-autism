# INT-0005 Bumetanide


Loop diuretic that reduces intracellular chloride in immature neurons, restoring GABA's inhibitory polarity. Phase 2 positive; phase 3 mixed.


**Dose range:** 0.5–1.0 mg twice daily


**Cost per month (USD):** 30


**Notes / warnings:** Lemonnier 2017 phase 2 positive

- [[MEC-0020 Calcium glutamate-NMDA homeostasis]]
