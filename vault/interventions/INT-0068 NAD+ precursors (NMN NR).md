# INT-0068 NAD+ precursors (NMN NR)


Nicotinamide mononucleotide / riboside; raise cellular NAD+. Activate sirtuins, support mitochondrial function. No autism RCT; promising aging biomarkers.


**Dose range:** 250-1000 mg/day


**Cost per month (USD):** 60
