# INT-0003 Methyl-B12 (methylcobalamin) injection


Provides active form of B12 to support methylation; subcutaneous injection bypasses GI absorption issues common in autism.


**Dose range:** 75 µg/kg every 3 days, SC


**Cost per month (USD):** 50


**Notes / warnings:** Hendren 2016 RCT
