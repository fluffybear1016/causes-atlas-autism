# INT-0087 Tryptophan supplementation (with vitamin D)


Patrick & Ames recommendation: tryptophan + vitamin D co-supplementation to support brain serotonin synthesis. Tryptophan crosses BBB via large neutral amino acid transporter; vitamin D upregulates TPH2 to use it.


**Dose range:** 500-2000 mg tryptophan + 2000 IU vit D /day


**Cost per month (USD):** 25


**Notes / warnings:** Speculative as autism intervention; mechanistic prior strong.
