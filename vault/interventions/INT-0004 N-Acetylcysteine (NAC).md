# INT-0004 N-Acetylcysteine (NAC)


Replenishes glutathione; modulates glutamatergic neurotransmission; documented effect on irritability in ASD trials.


**Dose range:** 900–2700 mg/day divided


**Cost per month (USD):** 25


**Notes / warnings:** Hardan 2012 RCT for irritability
