# BIO-0103 — Ferritin

**Category:** trace_minerals / iron_storage

**Sample type:** serum  
**Units:** ng/mL  
**Reference range:** 10–200  
**Age caveat:** Pediatric ranges differ substantially: infants 10-200 ng/mL; toddlers 7-140; school-age 10-150. >30 is functional optimal floor but normal infants may run lower.  

**What it measures:** Iron storage; common deficiency in autistic kids

**Low means:** Iron deficiency without anemia; common in restricted-diet ASD
