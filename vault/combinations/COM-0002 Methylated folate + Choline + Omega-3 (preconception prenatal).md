# COM-0002 Methylated folate + Choline + Omega-3 (preconception prenatal)


Strong individual-component evidence; combo not yet RCT-tested as such; biological rationale strong.


## Rationale

Methylation substrate stack covering folate (5-MTHF), choline (methyl donor), and DHA/EPA (membrane integrity + neuroinflammation). Each reduces autism risk individually in cohort data.


## Interaction warnings

None significant; standard prenatal vitamin caveats.
